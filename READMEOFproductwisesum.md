# Sales Total Program

## How to Run

1. Make sure you have Java installed on your computer

2. Download both files:
   - productwisesum.java
   - SalesEVESession.txt

3. Put both files in the SAME folder

4. Open the command prompt/terminal in that folder

5. Compile the program:
   javac productwisesum.java

6. Run the program:
   java productwisesum

## Input File Format

The SalesEVESession.txt file should be TAB-SEPARATED with these columns:
- ID
- Product Name
- Price
- Quantity

Example:
ID    Product    Price    Quantity
101    Laptop    50000    2
102    Mouse     500      5

## Output

The program will show the product-wise total sales amount.
